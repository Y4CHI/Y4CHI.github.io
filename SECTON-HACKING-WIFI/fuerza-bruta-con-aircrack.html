Fuerza bruta con Aircrack hacia hashes (handshakes)

La fuerza bruta con Aircrack hacia handshakes es una técnica utilizada para intentar descifrar contraseñas en una red Wi-Fi protegida mediante un ataque de fuerza bruta, específicamente orientado a los hashes de los handshakes capturados. Esto se hace con la intención de obtener la contraseña de la red Wi-Fi

El proceso general de este tipo de ataque involucra los siguientes pasos:

Captura de un handshake:

Un atacante primero debe capturar un handshake válido de la red Wi-Fi a la que desea acceder. El handshake se obtiene cuando un dispositivo legítimo se autentica o se reconecta a la red y se registra en un archivo de captura.

Extracción del hash:

A partir del handshake capturado, se extrae el hash de la contraseña que el atacante desea descifrar. El hash es una representación criptográfica de la contraseña original.

Ataque de fuerza bruta:

El atacante utiliza herramientas como Aircrack-ng para realizar un ataque de fuerza bruta contra el hash de la contraseña. Esto implica probar repetidamente diferentes contraseñas o combinaciones de caracteres hasta encontrar una que coincida con el hash.

Éxito o fracaso:

Si el ataque de fuerza bruta tiene éxito y se encuentra una contraseña que coincida con el hash, el atacante habrá obtenido acceso a la red Wi-Fi. Si el ataque no tiene éxito, el atacante puede intentar con otras contraseñas hasta encontrar la correcta o decidir abandonar el ataque.

Comencemos

aircrack-ng Es la herramienta a usar.

-w Es el parametro que sirve para especificar el archivo con la lista de contraseñas a usar.

captura-01.cap Es el archivo del handshake

Una vez que la herramienta llegue al 100% y hayamos tenido éxito, nos mostrará la contraseña o no, pero podemos intentar con múltiples listas de contraseñas hasta dar con la adecuada
