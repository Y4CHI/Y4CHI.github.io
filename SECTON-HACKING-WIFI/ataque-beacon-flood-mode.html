Ataque Beacon Flood Mode (Trama Beacon)

En este tipo de ataque, un atacante genera un gran número de tramas de tipo "beacon" y las envía de manera continua a una red Wi-Fi objetivo

Cosas Importantes:

Trama Beacon:

"beacon" es un tipo de trama de gestión en una red Wi-Fi que se emite periódicamente por un punto de acceso (router Wi-Fi) para anunciar su presencia y configuración a los dispositivos cercanos. Contiene información como el nombre de la red (SSID), el tipo de seguridad (por ejemplo, WPA2), y otros parámetros necesarios para que los dispositivos se conecten a la red.

Flood Mode:

En este contexto, "flood mode" significa que el atacante envía una gran cantidad de tramas Beacon al punto de acceso objetivo de manera repetitiva y continua, inundando la red con estas tramas.

Este es un tipo de ataque DOS que consta de saturar las redes

Generamos un fichero txt donde estará el nombre de los paquetes los cuales tendrán el nombre de las redes creadas, no es necesario crea este fichero para realizar el ataque, más adelante veremos como hacerlo sin el fichero de momento lo haremos mediante el fichero creado

Realizamos el ataque

b crea AP(puntos de acceso) falsos

-f esta opción especifica el archivo de texto que se utilizará como lista de redes para atacar. En este caso, el archivo se llama "redes.txt".

-a esta opción indica que se activará el modo de ataque automático, lo que significa que el programa elegirá automáticamente una red de la lista y la atacará sin necesidad de intervención manual.

-s esta opción establece la tasa de paquetes de balizas que se enviarán por segundo. En este caso, se han establecido 1000 paquetes por segundo, lo que indica que se enviarán muchas balizas por segundo para abrumar la red.

-c esta opción indica el canal que se utilizará para el ataque. En este caso, se ha seleccionado el canal 11, lo que significa que el ataque se llevará a cabo en ese canal específico.

Ahora lo haremos sin especificar los nombres y sin especificar el numero de paquetes, donde se crearan ESSID de forma aleatoria
