Ataque de Deautenticación global (Broadcast MAC Address)

Un "ataque de desautenticación global (Broadcast MAC Address)" en el contexto de las pruebas de penetración (pentesting) de redes WiFi se refiere a un ataque en el que un atacante envía paquetes de desautenticación a todas las estaciones (dispositivos) en una red WiFi utilizando una dirección MAC de difusión (broadcast MAC address).

La dirección MAC de difusión es una dirección especial que se utiliza para enviar paquetes a todas las estaciones en una red inalámbrica, en lugar de a una estación específica. Para garantizar el handshake podemos expulsar a todos con esta técnica.

aireplay-ng -0 10 -e JEXXXXX -c FF:FF:FF:FF:FF:FF wlan0mon

Podemos ver que se agrega la dirección MAC FF:FF:FF:FF:FF:FF y comienza a realizar ataques (Frames)

Ejecutamos el siguiente comando para hacer el ataque de deatenticación dirigido

Tambien se puede usar este comando para hacer lo mismo.

aireplay-ng -0 10 -e JEXXXXX wlan0mon
