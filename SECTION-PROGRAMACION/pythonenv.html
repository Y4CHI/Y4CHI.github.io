Como crear entornos virtuales con python

Porque es importante crear entornos virtuales con python ?

Trabajar en un entorno virtual , nos permite mejorar nuestra seguridad al ejecutar scripts , y tambien nos permite instalar las dependencias desde cero, para mejorar nuestra productividad , ya que nuestro entorno virtual estara sin librerias y solo las necesarias para funcionar.

Instalacion

        
    sudo apt-get update
    $sudo apt-get install python(version)

Paso 1

Creamos el entorno virtual y lo guardamos todas nuestras librerias y dependencias en el archivo .venv

             
     python3 -m venv .env

Paso 2

Despues de haber hecho lo anterior , debemos activar el entorno virtual y ya estara listo para trabajar, si desean desactivar el entorno virtual en la terminal : Deactivate

         
     source /route/activate

Paso 3

Despues de haber realizado todo lo anterior ,para comprobar si hemos realizado todo correctamente revisamos las librerias que tenemos con el siguiente comando: 

         
     pip list
     Package    Version
     ---------- -------
     pip        23.3
     setuptools 68.1.2

Si los resultados son algo parecido , ya se encontrara en entorno de trabajo virtual y podra realizar su proyecto sin problemas

Comandos extras

Crear un archivo con todas las dependencias para usarlo en otro momento

           
     pip freeze > requirements.txt

Para instalar dependencias necesarias de un archivo de texto del proyecto

           
      pip install -r requirements.txt
