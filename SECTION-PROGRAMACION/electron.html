Como crear una app de tu web favorita con electron

¿Qué es Electron?

Electron es un framework para crear aplicaciones de escritorio usando JavaScript, HTML y CSS. Incrustando Chromium y Node.js dentro del mismo, Electron le permite mantener una base de código JavaScript y crear aplicaciones multiplataforma que funcionan en Windows, macOS y Linux, - no requiere experiencia en desarrollo nativo.

Primeros pasos

Instalamos nodeJS usando nvm para instalar la version que gustamos instalar 

         
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash #Descargamos e instalamos nvm   
    nvm install --lts #Instalamos la version mas estable de nodeJS
    nvm use 16.20.2 #Afirmamos que vamos usar esa version exacta

Instalacion de Electron

         
    npm install --save-dev electron  #Instalamos Electron en nuestro Host
    npm install standard -D #Descargamos las dependencias necesarias para evitar errores futuros

Ya tenemos todo listo para poder comenzar a crear nuestra primera App Desktop

Crearemos un archivo package.json con los siguiente respectivo codigo 

        

        {
            "name": "web-app",
            "version": "1.0.0",
            "description": "",
            "main": "main.js",
            "scripts": {
              "dev" : "electron .",
              "test": "echo \"Error: no test specified\" && exit 1"
            },
            "keywords": [],
            "author": "",
            "license": "ISC",
            "devDependencies": {
              "electron": "^27.0.0",
              "standard": "17.1.0"
            },
          
            "eslintConfig": {
              "extends":"standard"
            }
        }

Si realizamos los primeros pasos que indicamos, ese archivo se creara automaticamente, si en caso no se creara, deberia de ser identicamente igual a lo mostrado.
Ahora nos toca crear nuestra archivo JavaScript con el nombre main.js para que detecte nuestra configuracion que acabamos de realizar.

Importamos el framework a nuestra archivo usando "const" el uso similar al from e import de python , en este caso llamamos a las funciones app, BrowserWindow e importamos Electron. 

            
  const { app, BrowserWindow } = require('electron')

Ahora creamos una variable llamada CreateWindow y hacemos el codigo para configurar el ancho y la alto que deseamos que tenga nuestra App , en este caso le dimos 800px de ancho y 600 de alto y luego dentro de "win.loadURL()" ingresamos la url que deseamos tener como una App en este caso sera https://www.cuevana3.eu/. 

             
        const createWindow = () => {
            const win = new BrowserWindow({
              width: 800,
              height: 600
            })
            win.loadURL('https://www.cuevana3.eu/')
        }

Ya tenemos ya casi todo listo para tener crear nuestra primera aplicacion de escritorio, lo que nos toca hacer Ahora es hacer que nos muestre en la pantalla, llamandolo y usando la funcion de App que nos traimos de Electron 

            
         app.whenReady().then(() => {
          createWindow()
         })

El codigo completo deberia de ser asi 

            
            const { app, BrowserWindow } = require('electron')

            const createWindow = () => {
                const win = new BrowserWindow({
                  width: 800,
                  height: 600
                  })
                  win.loadURL('https://www.cuevana3.eu/')
                }          
            app.whenReady().then(() => {
                  createWindow()
            })

Listo ahora tenemos todo preparado para poder correr nuestra primera app con el siguiente comando :

            
        npm run dev

Listo ya tendremos nuestra primera App Desktop creada , ahora lo podremos usar cuando nosotros deseamos , si desean simplemente probarlo se encuentra en nuestro github: https://github.com/Y4CHI/WebApp-Builder , lo pueden clonar e hacer los pasos que se le indican en el readme.md.
